# Finance Dashboard UI

## Overview

This is a frontend Finance Dashboard built using React and Tailwind CSS. It allows users to view financial summaries, transactions, and insights.

## Features

* Dashboard summary (Balance, Income, Expenses)
* Line chart (balance trends)
* Pie chart (spending breakdown)
* Transactions table with search and filter
* Role-based UI (Admin & Viewer)
* Add new transactions dynamically
* Data persistence using localStorage
* Insights section (expense analysis)
* Dark mode support 🌙
* Export transactions as JSON 📤
* Basic animations and responsive design

## Tech Stack

* React (Vite)
* Tailwind CSS
* Recharts

## How to Run

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`

## Notes

* This project uses mock data
* No backend is implemented
* Focus is on UI and frontend logic
